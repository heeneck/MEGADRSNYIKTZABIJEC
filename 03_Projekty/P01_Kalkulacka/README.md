# P01 — Kalkulačka ⭐

## Popis
Jednoduchá konzolová kalkulačka. Uživatel zadá dvě čísla a operaci (`+`, `-`, `*`, `/`), program vypočítá a vypíše výsledek. Obsahuje ošetření dělení nulou.

## Procvičovaná témata
- Vstup a převod na `double` (`Convert.ToDouble`)
- Podmínky `if / else if / else`
- Matematické operace
- Ošetření chybového stavu (dělení nulou)

## Ukázka
```
Zadej první číslo: 15
Vyber operaci: + - * /
Operace: *
Zadej druhé číslo: 4

15 * 4 = 60
```
