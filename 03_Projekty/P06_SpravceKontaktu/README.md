# P06 — Správce kontaktů ⭐⭐

## Popis
Konzolová aplikace s menu (přidat / zobrazit / hledat / ukončit). Kontakty (jméno + telefon) se ukládají do dvou propojených Listů.

## Procvičovaná témata
- Dva `List<string>` propojené přes stejný index
- `while` cyklus jako hlavní menu
- `for` cyklus s indexem pro přístup do obou Listů
- `.Contains()` pro vyhledávání (funguje i jako částečná shoda)
- `.Count` pro počet kontaktů
- Podmínky pro řízení menu

## Ukázka
```
--- MENU ---
1. Přidat kontakt
2. Zobrazit všechny kontakty
3. Vyhledat kontakt
4. Ukončit
Tvoje volba: 1
Jméno: Jana Nováková
Telefon: +420 777 123 456
✓ Kontakt 'Jana Nováková' přidán.
```
