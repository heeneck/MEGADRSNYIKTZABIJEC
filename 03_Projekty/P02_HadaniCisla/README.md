# P02 — Hádání čísla ⭐

## Popis
Klasická hra: program vygeneruje tajné číslo 1–100 a hráč hádá. Po každém tipu dostane nápovědu „víc" nebo „míň". Na konci se zobrazí počet pokusů a hodnocení.

## Procvičovaná témata
- Generování náhodných čísel (`Random.Shared.Next`)
- `while` cyklus s podmínkou na správný tip
- Podmínky `if / else if`
- Převod vstupu (`Convert.ToInt32`)
- Počítadlo pokusů

## Ukázka
```
Tvůj tip: 50
Moje číslo je VĚTŠÍ. Zkus víc!
Tvůj tip: 75
Moje číslo je MENŠÍ. Zkus míň!
Tvůj tip: 63

Správně! Číslo bylo 63.
Uhodl jsi na 3. pokus!
Skvělý výkon!
```
