# Nákupní seznam

## Co program dělá
Uživatel zadává položky nákupního seznamu jednu po druhé. Napsáním `konec` zadávání ukončí a program vypíše celý seznam s čísly.

## Procvičovaná témata
- `List<string>` a metoda `.Add()`
- `while` cyklus s podmínkou na vstup
- `foreach` cyklus pro výpis Listu
- Vlastnost `.Count` (počet prvků v Listu)

## Ukázka výstupu
```
=== Nákupní seznam ===
Zadávej položky. Pro ukončení napiš 'konec'.

Položka: Mléko
Položka: Chleba
Položka: Máslo
Položka: Jahody
Položka: konec

--- Tvůj nákupní seznam ---
Celkem 4 položek:

1. Mléko
2. Chleba
3. Máslo
4. Jahody
```

## Obtížnost
⭐⭐ Střední — List, while + foreach
