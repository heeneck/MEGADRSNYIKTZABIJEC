# Výpočet elektrického výkonu

## Co program dělá
Uživatel zadá napětí a proud, program vypočítá a vypíše elektrický výkon podle vzorce **P = U × I**.

## Procvičovaná témata
- Vstup od uživatele (`Console.ReadLine`)
- Převod textu na číslo (`Convert.ToInt32`)
- Matematické operace (násobení)
- Barvy konzole (`ForegroundColor`, `ResetColor`)

## Ukázka výstupu
```
=== Výpočet elektrického výkonu ===
Vzorec: P = U × I (výkon = napětí × proud)

Zadejte napětí (V):
> 230
Zadejte proud (A):
> 5
Výkon = 230 V × 5 A = 1150 Wattů
```
