1. Struktura dokumentu

<!DOCTYPE html>

Deklarace typu dokumentu. Říká prohlížeči, že jde o HTML5. Vždy na prvním řádku.

<!DOCTYPE html>

<html>

Kořenový element celé stránky. Atribut lang nastavuje jazyk.

<html lang="cs"> ... </html>

<head>

Obsahuje metadata — informace o stránce, které se nezobrazují přímo v obsahu.

<title>

Název stránky zobrazovaný v záložce prohlížeče a ve výsledcích vyhledávačů.

<title>Moje stránka</title>

<meta>

Metadata stránky — kódování, popis, klíčová slova, autor, viewport pro mobily.

<meta charset="UTF-8">
<meta name="description" content="Popis stránky">
<meta name="viewport" content="width=device-width, initial-scale=1.0">

<link>

Propojuje dokument s externími zdroji, nejčastěji CSS soubory nebo favicon.

<link rel="stylesheet" href="style.css">

<script>

Vkládá JavaScript — buď přímo nebo odkazem na soubor.

<script src="script.js"></script>

<style>

Vkládá CSS styly přímo do HTML dokumentu (interní CSS).

<style> body { color: red; } </style>

<body>

Obsahuje veškerý viditelný obsah stránky.

2. Nadpisy

<h1> až <h6>

Šest úrovní nadpisů. h1 je největší a nejdůležitější (měl by být jen jeden na stránce), h6 nejmenší.

h1 — Hlavní nadpis

h2 — Nadpis sekce

h3 — Podnadpis

h4 — Menší nadpis

h5 — Ještě menší
h6 — Nejmenší nadpis

3. Textové formátování

<p>

Odstavec textu. Základní blokový element pro text.

Toto je odstavec textu.

<br>

Zalomení řádku bez nového odstavce. Samouzavírací tag.

První řádek
Druhý řádek

<hr>

Horizontální oddělovací čára. Samouzavírací tag.

<strong> a <b>

<strong> — tučný text se sémantickým významem (důležitost). <b> — jen vizuálně tučný.

Důležitý text  |  Tučný text

<em> a <i>

<em> — kurzíva se sémantickým důrazem. <i> — jen vizuálně kurzíva.

Zdůrazněný text  |  Kurzíva

<u>

Podtržený text.

Podtržený text

<s> a <del>

<s> — přeškrtnutý text (neaktuální). <del> — smazaný text (editační změna).

Stará cena: 500 Kč  |  Smazaný text

<ins>

Vložený (nový) text, obvykle podtržený — protiklad k <del>.

Vložený text

<mark>

Zvýrazněný text (žluté pozadí jako zvýrazňovač).

Text s zvýraznením uprostřed.

<small>

Menší text — poznámky pod čarou, copyright apod.

Normální text a malý text

<sub> a <sup>

Dolní index (sub) a horní index (sup).

H2O  |  E = mc2

<blockquote>

Dlouhý citát z externího zdroje. Zobrazuje se jako odsazený blok.

"Kód by měl být psán tak, aby ho přečetl člověk, a jen náhodou i počítač."

<q>

Krátký inline citát. Prohlížeč automaticky přidá uvozovky.

Jak řekl Jobs: Stay hungry, stay foolish.

<abbr>

Zkratka nebo akronym. Atribut title zobrazí plný název při najetí myší.

HTML je základ webu.

<code>

Inline kód — zobrazí se neproporcionálním písmem.

Proměnná int x = 5; je celé číslo.

<pre>

Předformátovaný text — zachovává mezery a zalomení řádků. Vhodné pro bloky kódu.

int main() {
    printf("Hello, World!");
    return 0;
}

<kbd>

Klávesová zkratka nebo vstup z klávesnice.

Uložte soubor pomocí Ctrl + S

<span>

Inline kontejner bez sémantiky — slouží k aplikaci CSS nebo JavaScriptu na část textu.

Text s červenou barvou.

<div>

Blokový kontejner bez sémantiky — nejpoužívanější element pro rozvržení stránky.

Obsah div bloku

<address>

Kontaktní informace autora nebo vlastníka dokumentu.

Email: novak@example.com
Praha, Česká republika

4. Seznamy

<ul> — nečíslovaný seznam

Neuspořádaný seznam s odrážkami. Položky jsou <li>.

  • Jablko
  • Banán
  • Pomeranč

<ol> — číslovaný seznam

Uspořádaný seznam s čísly. Atributy type (1, A, a, I, i) a start.

  1. Otevři editor
  2. Napiš kód
  3. Ulož soubor

<dl> — definiční seznam

Seznam pojmů a jejich definic. <dt> = pojem, <dd> = definice.

5. Odkazy a obrázky

<a> — odkaz (anchor)

Hypertextový odkaz. Atribut href = cíl, target="_blank" = otevřít v nové záložce, rel="noopener" = bezpečnost.

Odkaz na Google
Odkaz na sekci na stránce (#kotva)
Email odkaz
Telefonní odkaz

<img> — obrázek

Vkládá obrázek. src = cesta k souboru, alt = alternativní text (povinný!), width a height = rozměry.


<img src="obrazek.jpg" alt="Popis obrázku" width="200" height="100">

<figure> a <figcaption>

Sémantický obal pro obrázek (nebo jiný media obsah) s popiskem.

8. Sémantické elementy (HTML5)

Sémantické elementy mají smysluplný název, který popisuje jejich obsah. Pomáhají vyhledávačům (SEO) i přístupnosti.

<header>

Záhlaví stránky nebo sekce — logo, navigace, nadpis. Na stránce může být více headerů.

<nav>

Navigační menu — obsahuje odkazy pro navigaci po stránce nebo webu.

<main>

Hlavní obsah stránky. Na stránce by měl být jen jeden <main>.

<section>

Tematická sekce dokumentu, obvykle s vlastním nadpisem.

<article>

Samostatný, přenositelný obsah — blogový příspěvek, novinový článek, komentář.

Titulek článku

Text článku, který by dával smysl i bez kontextu zbytku webu.

Autor: Jan Novák | 18. 5. 2026

<aside>

Vedlejší obsah — sidebar, poznámky, reklamy, související odkazy.

<footer>

Zápatí stránky nebo sekce — copyright, kontakty, sitemap.

<time>

Datum nebo čas v čitelném formátu. Atribut datetime je pro strojové zpracování.

Publikováno: 18. května 2026

9. Média

<audio>

Přehrávač zvuku. Atributy: controls (zobrazí ovládání), autoplay, loop, muted. Více formátů = fallback.

<audio controls>
  <source src="zvuk.mp3" type="audio/mpeg">
  <source src="zvuk.ogg" type="audio/ogg">
  Váš prohlížeč nepodporuje audio.
</audio>

<video>

Přehrávač videa. Stejné atributy jako audio + width, height, poster (náhledový obrázek).

<video controls width="400" poster="nahled.jpg">
  <source src="video.mp4" type="video/mp4">
  Váš prohlížeč nepodporuje video.
</video>

<iframe>

Vkládá jiný webový dokument nebo obsah (YouTube video, mapa) přímo do stránky.

<iframe src="https://www.youtube.com/embed/VIDEO_ID" width="560" height="315" allowfullscreen></iframe>
