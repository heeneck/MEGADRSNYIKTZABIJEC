# Z01 — Teploty v týdnu

## Zadání

Program uživatele vyzve, aby zadal teplotu pro každý den v týdnu, a to ve stylu:

```
Pondělí: 24.5
Úterý: 19.1
Středa: ...
...
```

Hodnoty se uloží do pole desetinných čísel (`double[]`), využít můžeš i `List<double>`.

Program poté vypíše:
- průměr teplot
- nejvyšší teplotu
- nejmenší teplotu

## Témata
- `double[]` nebo `List<double>` pro uložení 7 hodnot
- `for` cyklus pro načítání vstupů
- `Convert.ToDouble` pro převod textu na desetinné číslo
- výpočet průměru, hledání maxima a minima

## Ukázka výstupu
```
Pondělí: 24.5
Úterý: 19.1
Středa: 21.3
Čtvrtek: 18.7
Pátek: 22.0
Sobota: 25.1
Neděle: 23.4

Průměr teplot: 22.0 °C
Nejvyšší teplota: 25.1 °C
Nejmenší teplota: 18.7 °C
```
