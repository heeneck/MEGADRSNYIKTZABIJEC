# Generátor hesel

## Co program dělá
Program vygeneruje náhodné 8znakové heslo ze sady písmen (a–j) a číslic (0–9) pomocí pole a while cyklu.

## Procvičovaná témata
- Pole (`string[]`) a přístup přes index
- `while` cyklus (konečný s počítadlem)
- Generování náhodných čísel (`Random.Shared.Next`)
- Vlastnost `.Length` pro zjištění délky pole
- Skládání textu (`heslo = heslo + znak`)

## Ukázka výstupu
```
=== Generátor hesel ===
Generuji 8znakové heslo...
Vygenerované heslo: b3fh2ai9
```

*(každé spuštění dá jiný výsledek)*

## Obtížnost
⭐⭐ Střední — pole + Random + while
