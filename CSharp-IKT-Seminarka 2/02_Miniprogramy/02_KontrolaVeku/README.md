# Kontrola věku

## Co program dělá
Uživatel zadá číslo (věk) a program ho vypíše barevně jako: Zletilý, Nezletilý, nebo speciální reakce na extrémní hodnoty.

## Procvičovaná témata
- Vstup od uživatele (`Console.ReadLine`)
- Převod textu na číslo (`Convert.ToInt32`)
- Podmínky `if` / `else if` / `else`
- Barvy konzole (`ForegroundColor`, `ResetColor`)

## Ukázka výstupu
```
=== Kontrola věku ===
Zadejte věk:
> 15
Nezletilý
```

## Obtížnost
⭐ Lehká — vstup, podmínky, převod typu
