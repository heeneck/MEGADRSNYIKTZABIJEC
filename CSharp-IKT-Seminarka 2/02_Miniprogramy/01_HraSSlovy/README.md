# Hra se slovy a náhodnými čísly

## Co program dělá
Uživatel zadá `small` nebo `big` a program vygeneruje číslo v odpovídajícím rozsahu.

## Procvičovaná témata
- Vstup od uživatele (`Console.ReadLine`)
- Podmínky `if` / `else if` / `else`
- Generování náhodných čísel (`Random.Shared.Next`)

## Ukázka výstupu
```
=== Hra se slovy a čísly ===
Napiš 'small' nebo 'big':
> big
Zadal jsi 'big', tady je obrovské číslo: 487231
```

## Obtížnost
⭐ Lehká — základní podmínky a Random
