# Z03 — Chatbot (GJSBOT)

## Zadání

Program má 2× pole 15 frází — jedno pro věty, druhé pro otázky.
Uživatel zadá jméno, pak v nekonečném cyklu zadává věty.
Program 2 sekundy čeká, vyčistí obrazovku a odpoví:
- věta s `?` → náhodná fráze z pole 2
- ostatní věty → náhodná fráze z pole 1

```
Jonáš: Dneska jsem si koupil novou hopsakouli.
GJSBOT: To mě upřímně vůbec nepřekvapuje.
```

## Témata
- `string[]`, `while(true)`, `Contains("?")`, `Random`, `Thread.Sleep`, `Console.Clear`
