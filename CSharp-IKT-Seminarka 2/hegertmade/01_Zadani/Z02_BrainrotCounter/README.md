# Z02 — Brainrot counter

## Zadání

Uživatel zadává věty, dokud nenapíše `"STOP"`.
Pokud věta obsahuje brainrot slovíčko, uloží se do `List<string>`.
Po zadání STOP se vypíše obsah listu a počet vět:
```
Brainrot detekován v 8 větách
```

## Témata
- `List<string>`, `while`, `string.Contains`, `foreach`

## Brainrot slovíčka
`skibidi`, `rizz`, `sigma`, `gyatt`, `fanum`, `mewing`, `ohio`, `aura`, ...
