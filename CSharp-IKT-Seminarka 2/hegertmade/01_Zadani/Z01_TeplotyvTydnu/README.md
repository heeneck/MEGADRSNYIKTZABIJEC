# Z01 — Teploty v týdnu

## Zadání

Program uživatele vyzve, aby zadal teplotu pro každý den v týdnu:
```
Pondělí: 24.5
Úterý: 19.1
...
```
Hodnoty se uloží do pole `double[]` (nebo `List<double>`).
Program poté vypíše průměr teplot, nejvyšší a nejmenší teplotu.

## Témata
- `double[]` pro 7 hodnot
- `for` cyklus + `Convert.ToDouble`
- výpočet průměru, maxima, minima
