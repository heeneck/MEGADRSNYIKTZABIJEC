# P11 — Generátor násobilky ⭐

## Popis
Uživatel zadá číslo a program vypíše jeho násobilku přehledně zarovnanou do sloupce. Pak se nabídne volitelný procvičovací režim — 5 náhodných příkladů s okamžitým hodnocením.

## Procvičovaná témata
- `for` cyklus (1 až 10)
- Matematika (násobení)
- Vstup a `Convert.ToInt32`
- Zarovnání výpisu (`{cislo,2}` = zarovnání na 2 znaky)
- Podmínky pro hodnocení výsledku
- `Random.Shared.Next` v procvičovacím režimu

## Ukázka
```
Zadej číslo (1–20), jehož násobilku chceš vypsat: 7

  Násobilka čísla 7:
  ─────────────────────
  7 ×  1 =   7
  7 ×  2 =  14
  7 ×  3 =  21
  ...
  7 × 10 =  70
```
