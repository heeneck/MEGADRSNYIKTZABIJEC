# P12 — BMI Kalkulátor ⭐

## Popis
Uživatel zadá výšku (cm) a váhu (kg). Program vypočítá BMI podle vzorce WHO a zařadí výsledek do kategorie s barevným zvýrazněním a praktickými tipy.

## Vzorec
```
BMI = váha [kg] / (výška [m])²
```

## Procvičovaná témata
- `double` pro desetinná čísla
- `Convert.ToDouble` pro vstup
- Matematika s desetinnými čísly (dělení, mocnina)
- Podmínky `if / else if / else` s číselnými rozsahy
- Formátování výpisu (`{bmi:F1}` = 1 desetinné místo)
- Barvy konzole pro kategorie

## Ukázka
```
Zadej svou výšku v cm: 175
Zadej svou váhu v kg: 72

══════════════════════════════
Tvoje BMI: 23.5

Kategorie: Normální váha ✓
Výborně! Tvoje váha je v ideálním rozmezí.
```
